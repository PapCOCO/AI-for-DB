关于项目

                
            

            

                
DB for AI & AI for DB 是一个针对华为数据库实习生岗位设计的演示项目，探索数据库与AI融合的两个核心方向：

                
    
                    
  • AI for DB：用AI增强数据库功能（如NL2SQL、查询优化）
    • 
                    
  • DB for AI：用数据库支持AI应用（如向量存储、向量检索）
    • 
                

                
项目采用现代技术栈，包括Python、FastAPI、FAISS、ChromaDB等，提供了直观的Web界面，方便用户使用和测试。

            

            

                
            

        

    


    

        

            

                
使用帮助

                
            

            

                
NL2SQL 功能

                
1. 选择大语言模型（DeepSeek、OpenAI或HuggingFace）

                
2. 输入API Key（如果使用真实模型）

                
3. 在文本框中输入自然语言查询

                
4. 点击"生成SQL"按钮生成SQL语句

                
5. 点击"执行SQL"按钮执行查询并查看结果

                
                
向量数据库功能

                
1. 选择向量数据库类型（FAISS或ChromaDB）

                
2. 输入文档或上传文件

                
3. 点击"构建索引"按钮构建向量索引

                
4. 输入搜索查询并点击"搜索"按钮

                
5. 查看搜索结果和相似度分数

                
                
从数据库导入

                
1. 填写数据库连接信息

                
2. 点击"探索数据库"查看可用数据库

                
3. 选择数据库后，点击"探索表"查看表结构

                
4. 选择表后，点击"从数据库导入"按钮

                
                
用户登录

                
1. 点击右上角的"登录"按钮

                
2. 输入用户名和密码（测试账号：admin / password）

                
3. 登录后可以使用所有功能，包括执行SQL和向量数据库操作
